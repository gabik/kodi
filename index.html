Gabi's Kodi Repository

Go to http://kodi.tv/download/ and download Kodi (Kodi v17.1 "Krypton") for your OS.
Install it.
Run it.
Click on the Settings icon on the middle above the list (next to the shutdown)

Player Settings -> Language -> Language to download subtitles for -> Add Hebrew -> OK
back to settings (ESC)
System Settings -> Add-ons -> Enable Unknown sources
back to settings (ESC)
Interface Settings -> Skin -> Font -> Ariel
back to settings (ESC)
File Manager -> double click on "Add Source"
In the path enter: "http://kodi.kazav.net/repo"
In the Name enter "Gabi"

Click OK, and back to Main Menu (Several ESC)
Click on Add-Ons

Click on the box on the upper corner

Coose "Install from ZIP file"
Go to "Gabi" -> "Repository.Gabi" -> Choose the latest zip file -> OK
Wait 5 seconds for it to install
Choose "Install from repository"
Now you can see "Gabi Repository" and install whatever you want. Recommendations:
Video -> Exodus, Sdarot
Subtitles -> Torec, OpenSubtitles
To install you just need to choose the category (Videos for example), Click on the add-on name, and click install
If you have several sources for the same add-on choose the one from "Gabi repo" so you will stay auto updated

Some notes:
You can login to exodus with trakt.tv account to follow and manage your watchlists.
Logging in to Torec Subtitles will skip the waiting and some issue we have with guests users
To Login just goto Add-ons -> My Add-ons -> choose category (Videos/Subtitles) -> Choose the add on (Torec) -> Configure -> Fill the details

Done.
